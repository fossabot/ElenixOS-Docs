# ElenixOS System API

The `eos` namespace is the core system API provided by ElenixOS to the JavaScript script runtime. Through this API, applications and watch faces can access system services, page navigation, time information, configuration storage, console logs, and other functions.

## Console Logs (eos.console)

Provides different levels of log output for debugging and information recording.

### Methods

```javascript
eos.console.log(message)
eos.console.info(message)
eos.console.warn(message)
eos.console.error(message)
eos.console.debug(message)
```

**Parameters**:
- `message`: String type, the log information to output

**Return value**: None

**Description**: All log methods accept a string parameter and output it to the system log. When outputting, the current script ID is automatically added as a prefix to distinguish logs from different applications.

### Example

```javascript
eos.console.log("Application started");
eos.console.info("User logged in");
eos.console.warn("Low battery");
eos.console.error("Failed to load resource");
eos.console.debug("Position: x=" + x + ", y=" + y);
```

## View Management (eos.view)

Used to get the active View object of the current Activity, which is the foundation for creating UI components.

### eos.view.active()

```javascript
const view = eos.view.active();
```

**Parameters**: None

**Return value**: `lv_obj_t` type LVGL object, the View of the current Activity

**Description**: This method returns the LVGL object associated with the current Activity. All UI components should be created with this as the parent object. Equivalent to the C layer's `eos_view_active()`.

### Example

```javascript
// Get current View and create UI components
const view = eos.view.active();
const label = new lv.label(view);
label.setText("Hello, ElenixOS!");
label.center();

const button = new lv.button(view);
button.setSize(100, 50);
button.align(lv.ALIGN_CENTER, 0, 40);
```

## Activity Management (eos.activity)

Provides page navigation and Activity query functions. Activity is the page management unit of ElenixOS.

### Query Methods

#### eos.activity.current()

```javascript
const activity = eos.activity.current();
```

Gets the current Activity (top of the stack).

**Parameters**: None

**Return value**: `eos_activity_t` type Activity object, returns `undefined` on failure

#### eos.activity.visible()

```javascript
const activity = eos.activity.visible();
```

Gets the currently fully displayed (transition animation completed) Activity.

**Parameters**: None

**Return value**: `eos_activity_t` type Activity object, returns `undefined` on failure

#### eos.activity.bottom()

```javascript
const activity = eos.activity.bottom();
```

Gets the bottom Activity (usually the watch face Activity).

**Parameters**: None

**Return value**: `eos_activity_t` type Activity object, returns `undefined` on failure

#### eos.activity.watchface()

```javascript
const activity = eos.activity.watchface();
```

Gets the watch face Activity.

**Parameters**: None

**Return value**: `eos_activity_t` type Activity object, returns `undefined` on failure

#### eos.activity.rootScreen()

```javascript
const screen = eos.activity.rootScreen();
```

Gets the root screen object.

**Parameters**: None

**Return value**: `lv_obj_t` type root screen object

#### eos.activity.isTransitionInProgress()

```javascript
const isTransitioning = eos.activity.isTransitionInProgress();
```

Checks if Activity transition animation is in progress.

**Parameters**: None

**Return value**: `boolean` — `true` indicates transition animation in progress, `false` indicates idle

### View Management

#### eos.activity.getView(activity)

```javascript
const view = eos.activity.getView(activity);
```

Gets the View object of the specified Activity.

**Parameters**:
- `activity`: Activity object (returned by methods like `eos.activity.current()`)

**Return value**: `lv_obj_t` type View object, returns `undefined` on failure

#### eos.activity.setView(activity, view)

```javascript
eos.activity.setView(activity, view);
```

Sets the View object of the specified Activity.

**Parameters**:
- `activity`: Activity object
- `view`: LVGL object, as the Activity's View

**Return value**: None

### Title Management

#### eos.activity.getTitle(activity)

```javascript
const title = eos.activity.getTitle(activity);
```

Gets the Activity's title.

**Parameters**:
- `activity`: Activity object

**Return value**: `string` type, returns `undefined` on failure

#### eos.activity.setTitle(activity, title)

```javascript
eos.activity.setTitle(activity, title);
```

Sets the Activity's title.

**Parameters**:
- `activity`: Activity object
- `title`: String title

**Return value**: None

### Type Management

#### eos.activity.getType(activity)

```javascript
const type = eos.activity.getType(activity);
```

Gets the Activity's type.

**Parameters**:
- `activity`: Activity object

**Return value**: Number type, corresponds to `eos.ACTIVITY_TYPE_*` constants

#### eos.activity.setType(activity, type)

```javascript
eos.activity.setType(activity, type);
```

Sets the Activity's type.

**Parameters**:
- `activity`: Activity object
- `type`: Number type, use `eos.ACTIVITY_TYPE_*` constants

**Return value**: None

### AppHeader Visibility

#### eos.activity.setAppHeaderVisible(activity, visible)

```javascript
eos.activity.setAppHeaderVisible(activity, visible);
```

Sets the AppHeader visibility of the Activity.

**Parameters**:
- `activity`: Activity object
- `visible`: Boolean value

**Return value**: None

#### eos.activity.isAppHeaderVisible(activity)

```javascript
const visible = eos.activity.isAppHeaderVisible(activity);
```

Checks if the Activity's AppHeader is visible.

**Parameters**:
- `activity`: Activity object

**Return value**: `boolean`

### Navigation Methods

#### eos.activity.enter(activity)

```javascript
eos.activity.enter(activity);
```

Enters the specified Activity (pushes the Activity to the top of the stack and displays it).

**Parameters**:
- `activity`: Activity object to enter

**Return value**: None

#### eos.activity.back()

```javascript
eos.activity.back();
```

Returns to the previous Activity (destroys the current top Activity, restores the previous Activity).

**Parameters**: None

**Return value**: `boolean` — `true` indicates successful operation, `false` indicates failure

## Configuration Storage (eos.config)

Provides JSON file-based persistent key-value storage functionality, with each application/watch face having independent storage space.

**Storage paths**:
- Application: `<EOS_SYS_DIR>/app/app_data/<app_id>/config.json`
- Watch face: `<EOS_SYS_DIR>/app/watchface_data/<wf_id>/config.json`

### Write Methods

#### eos.config.setStr(key, value)

```javascript
eos.config.setStr("username", "Alice");
```

Stores a string value.

**Parameters**:
- `key`: String, key name
- `value`: String, value to store

**Return value**: None

#### eos.config.setBool(key, value)

```javascript
eos.config.setBool("dark_mode", true);
```

Stores a boolean value.

**Parameters**:
- `key`: String, key name
- `value`: Boolean value

**Return value**: None

#### eos.config.setNumber(key, value)

```javascript
eos.config.setNumber("volume", 75);
```

Stores a numeric value.

**Parameters**:
- `key`: String, key name
- `value`: Number

**Return value**: None

### Read Methods

#### eos.config.getStr(key)

```javascript
const username = eos.config.getStr("username");
```

Reads a string value. Returns `undefined` if the key does not exist.

**Parameters**:
- `key`: String, key name

**Return value**: `string` or `undefined`

#### eos.config.getBool(key)

```javascript
const darkMode = eos.config.getBool("dark_mode");
```

Reads a boolean value. Returns `false` if the key does not exist.

**Parameters**:
- `key`: String, key name

**Return value**: `boolean`

#### eos.config.getNumber(key)

```javascript
const volume = eos.config.getNumber("volume");
```

Reads a numeric value. Returns `0` if the key does not exist.

**Parameters**:
- `key`: String, key name

**Return value**: `number`

### Complete Example

```javascript
// Save user settings
eos.config.setStr("username", "Alice");
eos.config.setBool("dark_mode", true);
eos.config.setNumber("volume", 75);

// Read user settings
const username = eos.config.getStr("username");
const darkMode = eos.config.getBool("dark_mode");
const volume = eos.config.getNumber("volume");

eos.console.log("User: " + username + ", Dark Mode: " + darkMode + ", Volume: " + volume);
```

## Time Service (eos.time)

Provides functionality to get the current system time.

### eos.time.getNow()

```javascript
const now = eos.time.getNow();
```

**Parameters**: None

**Return value**: Time object with the following fields:

| Field | Type | Description | Range |
|-------|------|-------------|-------|
| `year` | number | Year | e.g., 2026 |
| `month` | number | Month | 1-12 |
| `day` | number | Day | 1-31 |
| `hour` | number | Hour | 0-23 |
| `min` | number | Minute | 0-59 |
| `sec` | number | Second | 0-59 |
| `ms` | number | Millisecond | 0-999 |
| `day_of_week` | number | Day of week | 0-6 (0=Sunday) |

### Example

```javascript
const now = eos.time.getNow();
eos.console.log("Current time: " + now.year + "-" + now.month + "-" + now.day + " " + now.hour + ":" + now.min + ":" + now.sec);

// Display time on watch face
const view = eos.view.active();
const timeLabel = new lv.label(view);

function updateTime() {
    const t = eos.time.getNow();
    const timeStr = String(t.hour).padStart(2, '0') + ":" + String(t.min).padStart(2, '0');
    timeLabel.setText(timeStr);
    timeLabel.center();
}
```

## Application Title Bar (eos.appHeader)

Controls the application's top navigation bar (AppHeader).

### eos.appHeader.setTitle(view, title)

```javascript
eos.appHeader.setTitle(view, "Settings");
```

Sets the application title bar title for the current Activity.

**Parameters**:
- `view`: View object or `null`
- `title`: Title string or `null`/`undefined`

**Return value**: None

### eos.appHeader.hide()

```javascript
eos.appHeader.hide();
```

Hides the application title bar.

**Parameters**: None

**Return value**: None

### eos.appHeader.show()

```javascript
eos.appHeader.show();
```

Shows the application title bar and refreshes the current Activity's title.

**Parameters**: None

**Return value**: None

### Example

```javascript
// Set title
eos.appHeader.setTitle(null, "My App");

// Hide title bar
eos.appHeader.hide();

// Show title bar
eos.appHeader.show();
```

## Watch Face Hands (eos.clockHand)

Creates and manages clock hands (hour, minute, second) for watch faces.

### eos.clockHand.create(obj, src, type, cx, cy)

```javascript
const hand = eos.clockHand.create(parent, "path/to/hand.png", eos.CLOCK_HAND_HOUR, cx, cy);
```

Creates a clock hand image.

**Parameters**:
- `obj`: Parent LVGL object
- `src`: String, pointer image resource path (relative to the application/watch face's assets directory)
- `type`: Hand type, use `eos.CLOCK_HAND_*` constants
- `cx`: Rotation center X coordinate (relative to the top-left corner of the image)
- `cy`: Rotation center Y coordinate (relative to the top-left corner of the image)

**Return value**: `lv_obj_t` type hand object

### eos.clockHand.center(obj)

```javascript
eos.clockHand.center(hand);
```

Aligns the center of the hand object to the center of its parent object.

**Parameters**:
- `obj`: Hand LVGL object

**Return value**: None

### eos.clockHand.placePivot(obj, x, y)

```javascript
eos.clockHand.placePivot(hand, cx, cy);
```

Sets the rotation center position of the hand object.

**Parameters**:
- `obj`: Hand LVGL object
- `x`: Rotation center X coordinate
- `y`: Rotation center Y coordinate

**Return value**: None

### Complete Example

```javascript
// Get current View
const view = eos.view.active();

// Create watch face background
const bg = new lv.image(view);
bg.setSrc("A:background.png");
bg.center();

// Create hour hand
const hourHand = eos.clockHand.create(view, "hour_hand.png", eos.CLOCK_HAND_HOUR, 5, 50);
eos.clockHand.center(hourHand);

// Create minute hand
const minHand = eos.clockHand.create(view, "min_hand.png", eos.CLOCK_HAND_MINUTE, 5, 65);
eos.clockHand.center(minHand);

// Create second hand
const secHand = eos.clockHand.create(view, "sec_hand.png", eos.CLOCK_HAND_SECOND, 3, 70);
eos.clockHand.center(secHand);
```

## System Constants

ElenixOS defines the following constants under the `eos` namespace for use in JavaScript scripts.

### Font Sizes

| Constant | Value | Description |
|----------|-------|-------------|
| `eos.FONT_SIZE_LARGE` | 30 | Large font |
| `eos.FONT_SIZE_MEDIUM` | 26 | Medium font |
| `eos.FONT_SIZE_SMALL` | 22 | Small font |

### Display Dimensions

| Constant | Value | Description |
|----------|-------|-------------|
| `eos.DISPLAY_WIDTH` | 240 | Screen width (pixels) |
| `eos.DISPLAY_HEIGHT` | 240 | Screen height (pixels) |

### Hand Types

| Constant | Value | Description |
|----------|-------|-------------|
| `eos.CLOCK_HAND_HOUR` | 0 | Hour hand |
| `eos.CLOCK_HAND_MINUTE` | 1 | Minute hand |
| `eos.CLOCK_HAND_SECOND` | 2 | Second hand |

### Activity Types

| Constant | Value | Description |
|----------|-------|-------------|
| `eos.ACTIVITY_TYPE_NULL` | 0 | Null type |
| `eos.ACTIVITY_TYPE_APP` | 1 | Application page |
| `eos.ACTIVITY_TYPE_APP_LIST` | 2 | Application list |
| `eos.ACTIVITY_TYPE_WATCHFACE` | 3 | Watch face page |
| `eos.ACTIVITY_TYPE_WATCHFACE_LIST` | 4 | Watch face list |

## Error Handling

### Error Types

Errors will be thrown when API call parameters are incorrect. Common error causes:

| Error Message | Reason |
|---------------|--------|
| `Invalid argument count` | Parameter count mismatch |
| `Invalid argument type` | Incorrect parameter type |
| `Usage: xxx(key, value)` | Usage hint, indicating the correct calling method |
| `Can't load config` | Configuration storage loading failed |
| `No current activity` | No active Activity |

### Error Handling Example

```javascript
try {
    const view = eos.view.active();
    if (view) {
        const label = new lv.label(view);
        label.setText("Hello!");
    }
} catch (err) {
    eos.console.error("Error: " + err);
}
```

## Complete Example: Counter Application

```javascript
// Get current View
const view = eos.view.active();

// Set title
eos.appHeader.setTitle(null, "Counter");

// Read saved count
let count = eos.config.getNumber("count");

// Create display label
const countLabel = new lv.label(view);
countLabel.setStyleTextFont(eos.FONT_SIZE_LARGE, 0);
countLabel.setText(String(count));
countLabel.align(lv.ALIGN_CENTER, 0, -30);

// Create increment button
const incBtn = new lv.button(view);
incBtn.setSize(80, 50);
incBtn.align(lv.ALIGN_CENTER, -50, 30);
incBtn.addEventCb(function() {
    count++;
    countLabel.setText(String(count));
    eos.config.setNumber("count", count);
}, lv.EVENT_CLICKED, null);

const incLabel = new lv.label(incBtn);
incLabel.setText("+1");

// Create decrement button
const decBtn = new lv.button(view);
decBtn.setSize(80, 50);
decBtn.align(lv.ALIGN_CENTER, 50, 30);
decBtn.addEventCb(function() {
    count--;
    countLabel.setText(String(count));
    eos.config.setNumber("count", count);
}, lv.EVENT_CLICKED, null);

const decLabel = new lv.label(decBtn);
decLabel.setText("-1");
```

## Complete Example: Digital Watch Face

```javascript
function updateWatchface() {
    const now = eos.time.getNow();

    // Format time
    const timeStr = String(now.hour).padStart(2, '0') + ":" +
                    String(now.min).padStart(2, '0');

    // Format date
    const dateStr = now.year + "-" +
                    String(now.month).padStart(2, '0') + "-" +
                    String(now.day).padStart(2, '0');

    // Update UI
    timeLabel.setText(timeStr);
    dateLabel.setText(dateStr);

    // Update hand angles
    const hourAngle = (now.hour % 12) * 30 + now.min * 0.5;
    const minAngle = now.min * 6 + now.sec * 0.1;
    const secAngle = now.sec * 6;
}

// Get View and create UI
const view = eos.view.active();
eos.appHeader.hide();

// Create time label
const timeLabel = new lv.label(view);
timeLabel.setStyleTextFont(eos.FONT_SIZE_LARGE, 0);
timeLabel.align(lv.ALIGN_CENTER, 0, -20);

// Create date label
const dateLabel = new lv.label(view);
dateLabel.setStyleTextFont(eos.FONT_SIZE_SMALL, 0);
dateLabel.align(lv.ALIGN_CENTER, 0, 10);

// Update every second
const timer = new lv.timer(function() {
    updateWatchface();
}, 1000, null);
updateWatchface();
```

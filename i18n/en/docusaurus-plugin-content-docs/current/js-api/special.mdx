# Special API Usage and Notes

ElenaOS provides some special LVGL API components that have specific functions and usage methods. This document will introduce the usage methods and notes of these special APIs.

## Special API Component List

The following are special LVGL API components supported by ElenaOS:

- [Animation](./widgets/anim) - Used to create and manage animation effects
- [Button Matrix](./widgets/btnm) - Used to create button matrices
- [Calendar](./widgets/calendar) - Used to display and select dates
- [Canvas](./widgets/canvas) - Used to draw graphics and images
- [Dropdown](./widgets/dropdown) - Used to display and select dropdown options
- [Image](./widgets/image) - Used to display images
- [Object](./widgets/obj) - Base class for all LVGL components
- [Timer](./widgets/timer) - Used to create and manage timers

## Features of Special APIs

Compared with ordinary APIs, special APIs have the following features:

1. **Specific functionality**: Each special API component has specific functionality, such as animation, calendar, etc.
2. **Complex usage**: The usage methods of special API components are usually more complex than ordinary components
3. **Performance requirements**: Some special API components have higher performance requirements, such as animation and canvas
4. **Resource consumption**: Some special API components may consume more system resources, such as canvas and image

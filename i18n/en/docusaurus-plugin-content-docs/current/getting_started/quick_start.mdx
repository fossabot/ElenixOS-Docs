# Quick Start

This guide will help you quickly get started with the ElenixOS project, including code acquisition, environment setup, compilation and running, and basic usage.

[ElenixOS-Simulator](https://github.com/ElenixOS/ElenixOS-Simulator) is based on [lv_port_pc_vscode](https://github.com/lvgl/lv_port_pc_vscode) and ported to ElenixOS,
allowing you to compile and run ElenixOS simulation on any PC for rapid iteration and debugging.

## Choose Your Starting Method

Based on your needs and environment, choose one of the following methods to start using ElenixOS:

### 1. Online Experience

If you want to quickly experience ElenixOS without installing any software, you can use the online simulator:

- [Online Simulator](https://simulator.elenixos.com/wasm/latest/main.html) - Experience ElenixOS directly in your browser without installing any software

### 2. Local Development

If you plan to develop ElenixOS applications or customize the system, it is recommended to set up a local development environment:

1. **Environment Setup** - Install necessary development tools and dependencies
   - [Environment Setup](./environment.mdx) - Detailed environment setup guide

2. **Build Project** - Compile and run the ElenixOS simulator
   - [Build Guide](./build.mdx) - Complete build process and command instructions

3. **Develop Applications** - Understand ElenixOS core concepts and APIs
   - [File System Module](../core/fs.mdx) - File operations and directory management
   - [Activity and View](../core/activity.mdx) - Page management and lifecycle
   - [JavaScript API](../js-api/elenix_os.mdx) - System API reference

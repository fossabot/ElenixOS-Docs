# 快速开始

本指南将帮助你快速上手 ElenixOS 项目，包括代码获取、环境搭建、编译运行和基本使用。

[ElenixOS-PC-Simulator](https://github.com/ElenixOS/ElenixOS-PC-Simulator) 基于 [lv_port_pc_vscode](https://github.com/lvgl/lv_port_pc_vscode) 对 ElenixOS 进行了移植，
您可以在任意 PC 上编译并运行模拟 ElenixOS，以便快速迭代和调试。

# 快速开始

本指南将帮助你快速上手 ElenixOS 项目，包括代码获取、环境搭建、编译运行和基本使用。

[ElenixOS-Simulator](https://github.com/ElenixOS/ElenixOS-Simulator) 基于 [lv_port_pc_vscode](https://github.com/lvgl/lv_port_pc_vscode) 对 ElenixOS 进行了移植，
您可以在任意 PC 上编译并运行模拟 ElenixOS，以便快速迭代和调试。

## 选择开始方式

根据您的需求和环境，选择以下方式之一开始使用 ElenixOS：

### 1. 在线体验

如果您想快速体验 ElenixOS 而不需要安装任何软件，可以使用在线模拟器：

- [在线模拟器](https://simulator.elenixos.com/wasm/latest/main.html) - 在浏览器中直接体验 ElenixOS，无需安装任何软件

### 2. 本地开发

如果您计划进行 ElenixOS 应用开发或系统定制，建议在本地搭建开发环境：

1. **环境搭建** - 安装必要的开发工具和依赖
   - [环境设置](/docs/getting_started/environment.mdx) - 详细的环境搭建指南

2. **构建项目** - 编译和运行 ElenixOS 模拟器
   - [构建指南](/docs/getting_started/build.mdx) - 完整的构建流程和命令说明

3. **开发应用** - 了解 ElenixOS 的核心概念和 API
   - [文件系统模块](architecture/services/storage/storage) - 文件操作和目录管理
   - [Activity 和 View](architecture/framework/activity/activity) - 页面管理和生命周期
   - [JavaScript API](architecture/script_engine/elenix_os) - 系统 API 参考

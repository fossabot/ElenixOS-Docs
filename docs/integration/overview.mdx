## 项目获取

### 克隆代码库

使用 Git 克隆 ElenaOS 代码库：

```bash
git clone --recursive https://github.com/Sab1e-dev/ElenaOS.git
```

:::note
注意：使用 `--recursive` 参数确保同时克隆子模块。
如果没有使用 `--recursive` 克隆，可以通过以下命令获取子模块：

```bash
git submodule update --init --recursive
```
:::

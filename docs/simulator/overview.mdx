# 模拟器概述

ElenixOS 提供了 PC 模拟器，基于 [lv_port_pc_vscode](https://github.com/lvgl/lv_port_pc_vscode) 对 ElenixOS 进行了移植，
您可以在任意 PC 上编译并运行模拟 ElenixOS，以便快速迭代和调试。

本页面将介绍如何获取代码、搭建环境、编译运行以及基本使用方法。

## 模拟器运行方式

### 本地运行（Native Simulator）

本地运行是指在您的计算机上直接编译和运行 ElenixOS 模拟器，支持 Linux、macOS 和 Windows 操作系统。

[查看本地运行指南](./native)

### 网页运行（WASM Simulator）

网页运行是指通过浏览器访问 ElenixOS 模拟器，无需安装任何依赖，即可直接体验 ElenixOS。

[查看网页运行指南](./wasm)
